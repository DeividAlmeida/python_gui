
# Sugestões de Designações

Esse é um aplicativo desktop em python cujo objetivo é consumir uma api e gerar em xlsx uma sugestão de designações. 


## Instalação

Instale as seguintes bibliotecas/frameworks com o pip

```bash
  pip install flet pandas openpyxl
```
    
## Funcionalidades

- Listar participantes
- Criar novos participantes
- Editar participantes
- Deletar participantes
- Gerar lista de designações em xlsx


## Stack utilizada

**GUI:** Python 3.10.12, Flet 0.9.0, Pandas 2.1.2
